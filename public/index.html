ZONA PRUEBAS API

Documentación de la API

Bienvenido a la documentación oficial de la API de GOAD. A continuación encontrarás el detalle de los endpoints disponibles, los métodos soportados, y el formato de los datos que debes enviar.

Autenticación y Usuarios

POST /api/usuarios

Registra un nuevo usuario en el sistema. Las contraseñas se guardan encriptadas con bcrypt.

Cuerpo (JSON):
Campo Tipo Descripción
dni String (15) Documento de identidad único (Requerido)
nombres String Nombres del usuario (Requerido)
apellido_paterno String Apellido paterno (Requerido)
email String Correo electrónico único (Requerido)
passw String Contraseña en texto plano (Requerido)
rol String Debe ser admin o cliente (Requerido)
POST /api/auth/login

Autentica a un usuario y devuelve un token JWT.

Cuerpo (JSON): 
{
  "email": "correo@ejemplo.com",
  "passw": "micontrasena123"
}
Respuesta (200 OK): 
{
  "token": "eyJhbGciOiJIUzI1NiIs...",
  "user": { "id": 1, "nombres": "Juan", "email": "..." }
}
Uso del Token: Para hacer peticiones protegidas (ej. crear anuncios), debes enviar este token en los Headers de tu petición HTTP:
Authorization: Bearer [TU_TOKEN]

Anuncios Base

GET /api/anuncios

Devuelve todos los anuncios activos (que no están eliminados lógicamente), cruzando datos con el autor y la categoría.

POST /api/anuncios

Crea un anuncio básico sin atributos específicos de vehículos ni inmuebles.

Cuerpo (JSON):
Campo Tipo Descripción
titulo String Título descriptivo (Requerido)
descripcion String Texto descriptivo (Requerido)
latitud Number Ej. -12.043180 (Opcional, máx 2 dígitos enteros)
longitud Number Ej. -77.028240 (Opcional, máx 3 dígitos enteros)
numero_telefono String (Opcional)
id_usuario Integer ID del autor (Requerido)
id_categoria Integer ID de la categoría general (Requerido)

Vehículos (Anuncios Compuestos)

POST /api/vehiculos

Crea internamente un Anuncio Base, detecta la categoría y en la misma transacción inserta los detalles técnicos del vehículo.

Cuerpo (JSON):

Se envía un solo JSON mezclando la data del anuncio y del vehículo:

{
  "titulo": "Vendo Mazda 3",
  "descripcion": "Excelente estado",
  "id_usuario": 1,
  "id_categoria": 2,

  "precio": 14500.50,
  "marca": "Mazda",
  "modelo": "3",
  "anio": 2021,
  "kilometraje": 45000,
  "combustible": "Gasolina",
  "airbag": true,
  "aire": true,
  "sunroof": false
}

Imágenes de Anuncios

POST /api/anuncios/:id/imagenes

Sube una imagen física al disco D:/imagenes usando Multer y vincula su ruta relativa en la base de datos.

Formato: multipart/form-data
Key Valor
imagen Archivo físico (.jpg, .png, .webp)
GET /api/anuncios/:id/imagenes

Obtiene un array de objetos con las URLs estáticas de las fotos de un anuncio.
